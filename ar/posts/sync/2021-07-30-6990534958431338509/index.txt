<p><strong>2.v-model底层是怎么实现的？</strong></p>
<p><code>v-model</code>
会把关联的相应式数据(info.message)，动态的绑定到表单元素的value属性上，让后监听
<code>input</code> 事件；</p>
<p>当 <code>v-model</code>
绑定的相应数据发生变化时，表单元素的value值也会随之变化。</p>
</blockquote>
<pre class="vue"><code>&lt;template&gt;
  &lt;div&gt;
    &lt;div class=&quot;message&quot;&gt;{{ info.message }}&lt;/div&gt;
    &lt;div&gt;&lt;input v-model=&quot;info.message&quot; type=&quot;text&quot;&gt;&lt;/div&gt;
    &lt;button @click=&quot;change&quot;&gt;click&lt;/button&gt;
  &lt;/div&gt;
&lt;/template&gt;

&lt;script&gt;
  export default {
    data () {
      return {
        info: {}
      }
    },
    methods: {
      change () {
        this.info.message = &#39;hello world&#39;
      }
    }
  }
&lt;/script&gt;</code></pre>

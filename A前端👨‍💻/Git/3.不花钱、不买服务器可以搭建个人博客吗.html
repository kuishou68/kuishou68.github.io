    # 我想把它映射到自己的域名可以嘛？

    没问题的！你只需要花几十块钱买个域名！

    首先，你得注册一个属于你自己的域名，这边我用的华为云,其他平台阿里云、腾讯云、百度云都可以，只要能买域名就行。域名注册 -> 域名备案 -> 域名解析，备案需要等待5-15天才能备案成功，这段时间你可以先好好整理一下自己的博客内容。

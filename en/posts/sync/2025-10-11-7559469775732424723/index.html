дёҖгҖҒдј иҫ“зҡ„жң¬иҙЁ

1.В Stdio (Standard Input/Output)

еҚҸи®®жң¬иҙЁпјҡ

  • еҹәдәҺВ иҝӣзЁӢй—ҙйҖҡдҝЎпјҲIPCпјү В зҡ„з»Ҹе…ёжЁЎејҸ
  • еҲ©з”Ё Unix/Linux зҡ„з®ЎйҒ“жңәеҲ¶пјҲpipeпјү
  • ж¶ҲжҒҜж јејҸпјҡJSON-RPC 2.0 over newline-delimited JSON

е·ҘдҪңжөҒзЁӢпјҡ

MCP Server (Child Process)-->>Client: stdout: {"result":{...}}

MCP Server (Child Process)-->>Client: stdout: {"result":{...}}

дҪҝз”ЁеңәжҷҜпјҡ

  • вң… жң¬ең°е‘Ҫд»ӨиЎҢе·Ҙе…·пјҲеҰӮВ npx,В python,В nodeпјү
  • вң… ејҖеҸ‘иҖ…е·Ҙе…·пјҲLSPгҖҒformatterгҖҒlinterпјү
  • вң… зҰ»зәҝеңәжҷҜпјҲж— зҪ‘з»ңдҫқиө–пјү
  • вң… йңҖиҰҒзҺҜеўғеҸҳйҮҸйҡ”зҰ»зҡ„е·Ҙе…·

дјҳзӮ№пјҡ

  1. йӣ¶зҪ‘з»ңејҖй”ҖпјҡзӣҙжҺҘиҝӣзЁӢйҖҡдҝЎпјҢе»¶иҝҹ < 1ms
  2. е®үе…ЁжҖ§й«ҳпјҡдёҚжҡҙйңІз«ҜеҸЈпјҢдёҚеҸ—зҪ‘з»ңж”»еҮ»
  3. з®ҖеҚ•зӣҙжҺҘпјҡspawn дёҖдёӘиҝӣзЁӢе°ұиғҪз”Ё
  4. иө„жәҗжҺ§еҲ¶зІҫзЎ®пјҡеҸҜд»ҘйҡҸж—¶ kill иҝӣзЁӢ

зјәзӮ№пјҡ

  1. еҸӘиғҪжң¬ең°пјҡж— жі•и·ЁжңәеҷЁи°ғз”Ё
  2. иҝӣзЁӢеҗҜеҠЁејҖй”ҖпјҡжҜҸж¬Ўи°ғз”ЁеҸҜиғҪйңҖиҰҒеҗҜеҠЁж–°иҝӣзЁӢпјҲеҰӮ npx йңҖиҰҒдёӢиҪҪдҫқиө–пјү
  3. ж— жі•е…ұдә«е®һдҫӢпјҡжҜҸдёӘе®ўжҲ·з«ҜйғҪеҗҜеҠЁзӢ¬з«ӢиҝӣзЁӢпјҢжөӘиҙ№иө„жәҗ
  4. и°ғиҜ•еӣ°йҡҫпјҡй”ҷиҜҜдҝЎжҒҜеҸӘиғҪд»Һ stderr иҺ·еҸ–

е…ёеһӢе®һзҺ°пјҡ

  • Claude Desktop зҡ„жң¬ең° MCP е·Ҙе…·
  • VS Code Extension зҡ„ Language Server Protocol (LSP)
  • Git зҡ„В filter-branchВ зӯүз®ЎйҒ“е‘Ҫд»Ө

2.В SSE (Server-Sent Events)

еҚҸи®®жң¬иҙЁпјҡ

  • еҹәдәҺ HTTP зҡ„В еҚ•еҗ‘жҺЁйҖҒВ еҚҸи®®
  • жңҚеҠЎеҷЁдё»еҠЁеҗ‘е®ўжҲ·з«ҜеҸ‘йҖҒдәӢд»¶жөҒ
  • еә•еұӮжҳҜ HTTP й•ҝиҝһжҺҘпјҲContent-Type: text/event-streamпјү

е·ҘдҪңжөҒзЁӢпјҡ

  • вң… жңҚеҠЎеҷЁзҠ¶жҖҒжӣҙж–°пјҲеҰӮ MCP е·Ҙе…·еҲ—иЎЁеҸҳжӣҙпјү
  • вң… иҝӣеәҰеҸҚйҰҲпјҲеҰӮй•ҝж—¶й—ҙд»»еҠЎзҡ„иҝӣеәҰжқЎпјү
  • вң… йңҖиҰҒиҮӘеҠЁйҮҚиҝһзҡ„еңәжҷҜпјҲжөҸи§ҲеҷЁеҺҹз”ҹж”ҜжҢҒпјү

дјҳзӮ№пјҡ

  1. иҮӘеҠЁйҮҚиҝһпјҡж–ӯзәҝеҗҺжөҸи§ҲеҷЁдјҡиҮӘеҠЁйҮҚж–°иҝһжҺҘ
  2. иҪ»йҮҸзә§пјҡжҜ” WebSocket з®ҖеҚ•пјҢйҖӮеҗҲеҚ•еҗ‘жҺЁйҖҒ
  3. и·ЁеҹҹеҸӢеҘҪпјҡж”ҜжҢҒ CORS
  4. дәӢд»¶ ID жңәеҲ¶пјҡж”ҜжҢҒж–ӯзәҝз»ӯдј пјҲLast-Event-IDпјү

зјәзӮ№пјҡ

  1. еҚ•еҗ‘йҖҡдҝЎпјҡеҸӘиғҪжңҚеҠЎеҷЁ вҶ’ е®ўжҲ·з«ҜпјҢе®ўжҲ·з«ҜеҸ‘иҜ·жұӮйңҖиҰҒйўқеӨ– HTTP
  2. жөҸи§ҲеҷЁйҷҗеҲ¶пјҡжңүдәӣжөҸи§ҲеҷЁеҜ№еҗҢеҹҹ SSE иҝһжҺҘж•°жңүйҷҗеҲ¶пјҲеҰӮ 6 дёӘпјү
  3. д»ЈзҗҶй—®йўҳпјҡжҹҗдәӣдјҒдёҡд»ЈзҗҶдјҡзј“еҶІ SSE жөҒ
  4. дёҚйҖӮеҗҲеҸҢеҗ‘дәӨдә’пјҡеҰӮжһңйңҖиҰҒйў‘з№ҒеҸ‘йҖҒиҜ·жұӮпјҢдёҚеҰӮз”Ё WebSocket

е…ёеһӢе®һзҺ°пјҡ

  • ChatGPT зҡ„жөҒејҸиҫ“еҮәпјҲж—©жңҹдҪҝз”Ё SSEпјү
  • GitHub Actions зҡ„ж—Ҙеҝ—жөҒ
  • иӮЎзҘЁиЎҢжғ…жҺЁйҖҒ

  • вң… еӨҡе®ўжҲ·з«Ҝе…ұдә«еҗҢдёҖжңҚеҠЎпјҲеҰӮж•°жҚ®еә“гҖҒзҹҘиҜҶеә“пјү
  • вң… йңҖиҰҒиҙҹиҪҪеқҮиЎЎгҖҒCDNгҖҒзј“еӯҳзӯүеҹәзЎҖи®ҫж–Ҫ

дјҳзӮ№пјҡ

  1. з”ҹжҖҒжҲҗзҶҹпјҡHTTP е·Ҙе…·й“ҫе®Ңе–„пјҲnginxгҖҒHAProxyгҖҒCDNпјү
  2. жҳ“дәҺйғЁзҪІпјҡж ҮеҮҶ Web жңҚеҠЎпјҢе®№еҷЁеҢ–гҖҒK8s еҸӢеҘҪ
  3. ж— зҠ¶жҖҒпјҡжҜҸдёӘиҜ·жұӮзӢ¬з«ӢпјҢжҳ“дәҺж°ҙе№іжү©еұ•
  4. жҳ“дәҺи°ғиҜ•пјҡcurlгҖҒPostmanгҖҒжөҸи§ҲеҷЁйғҪиғҪжөӢиҜ•
  5. йҳІзҒ«еўҷеҸӢеҘҪпјҡеҮ д№ҺжүҖжңүзҪ‘з»ңзҺҜеўғйғҪе…Ғи®ё HTTP

зјәзӮ№пјҡ

  1. иҝһжҺҘејҖй”ҖеӨ§пјҡжҜҸж¬ЎиҜ·жұӮйғҪйңҖиҰҒ TCP дёүж¬ЎжҸЎжүӢгҖҒTLS жҸЎжүӢ
  2. е»¶иҝҹй«ҳпјҡзӣёжҜ”жң¬ең° IPCпјҢзҪ‘з»ңе»¶иҝҹ 10-100ms
  3. ж— жі•дё»еҠЁжҺЁйҖҒпјҡжңҚеҠЎеҷЁж— жі•дё»еҠЁйҖҡзҹҘе®ўжҲ·з«ҜпјҲйңҖиҰҒиҪ®иҜўжҲ– webhookпјү
  4. дёҚйҖӮеҗҲй«ҳйў‘и°ғз”ЁпјҡжҜҸз§’дёҠеҚғж¬ЎиҜ·жұӮдјҡеҫҲдҪҺж•Ҳ

е…ёеһӢе®һзҺ°пјҡ

  • RESTful APIпјҲеҰӮ GitHub APIгҖҒStripe APIпјү

  • вң… еӨ§ж–Үд»¶дј иҫ“пјҲеҲҶеқ—жөҒејҸеҸ‘йҖҒпјү
  • вң… жңүзҠ¶жҖҒзҡ„дјҡиҜқз®ЎзҗҶпјҲеҰӮеӨҡиҪ®еҜ№иҜқпјү

дјҳзӮ№пјҡ

  1. еҸҢеҗ‘иғҪеҠӣпјҡиҜ·жұӮз”Ё HTTPпјҢе“Қеә”еҸҜд»ҘжөҒејҸ
  2. дјҡиҜқз®ЎзҗҶпјҡйҖҡиҝҮ session ID з»ҙжҠӨдёҠдёӢж–Ү
  3. жёҗиҝӣејҸе“Қеә”пјҡйҖӮеҗҲз”ҹжҲҗејҸ AI зҡ„жөҒејҸиҫ“еҮә
  4. е…је®№жҖ§еҘҪпјҡеҹәдәҺж ҮеҮҶ HTTP/SSEпјҢж— йңҖзү№ж®ҠеҚҸи®®

  1. еӨҚжқӮеәҰй«ҳпјҡйңҖиҰҒжңҚеҠЎз«Ҝз®ЎзҗҶ session зҠ¶жҖҒ
  2. еҶ…еӯҳејҖй”ҖпјҡжңҚеҠЎз«ҜйңҖиҰҒеӯҳеӮЁдјҡиҜқдҝЎжҒҜ
